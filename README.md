# Tic Tac Toe Game 🎮

This is a simple browser-based Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## Features
- Two-player gameplay
- Win detection logic
- Draw detection
- Reset game functionality
- Interactive user interface

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation & Event Handling)

## How to Play
1. Open the game in a browser.
2. Two players take turns marking X and O.
3. The first player to align three symbols wins.
4. If all cells are filled without a winner, the game ends in a draw.

## Project Purpose
This project was created to practice **JavaScript logic, DOM manipulation, and event handling**.
